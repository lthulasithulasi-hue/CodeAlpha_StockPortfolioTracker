# Stock Portfolio Tracker

A simple Python application that helps you track and manage your stock portfolio by calculating investment values and generating reports.

## Features

- **Stock Price Database**: Pre-defined prices for major stocks (AAPL, TSLA, GOOGL, MSFT, AMZN)
- **Portfolio Management**: Add multiple stocks to your portfolio with custom quantities
- **Investment Calculation**: Automatically calculates total investment value for each stock
- **Portfolio Summary**: Displays a summary of all holdings and their current values
- **Report Generation**: Exports portfolio details to a text file (`portfolio_report.txt`)

## Requirements

- Python 3.x
